# Duarte Execução de Projetos

Este é um projeto de uma página de contato desenvolvido para a empresa **Duarte Execução de Projetos**, com o objetivo de oferecer uma forma rápida e intuitiva para que clientes entrem em contato por diferentes canais de atendimento.

A página apresenta um layout moderno e responsivo, com destaque para os principais meios de comunicação da empresa, incluindo telefone, WhatsApp, Instagram e e-mail. Além disso, o projeto possui uma mensagem de boas-vindas personalizada, exibida após o visitante informar seu nome.

## Funcionalidades

* Acesso rápido ao telefone da empresa.
* Contato direto pelo WhatsApp.
* Link para o perfil oficial no Instagram.
* Área destinada ao contato por e-mail.
* Mensagem de boas-vindas personalizada utilizando JavaScript.
*Interface moderna desenvolvida com HTML e CSS.
*Ícones profissionais utilizando Font Awesome.

## Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* Font Awesome

## Objetivo

Este projeto foi desenvolvido para praticar conceitos de desenvolvimento Front-end, como estruturação de páginas com HTML, estilização com CSS e interatividade utilizando JavaScript, ao mesmo tempo em que atende à necessidade de divulgação dos canais de contato da empresa.
